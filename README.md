# Apache Kylin (apache-kylin)
Apache Kylin is an open-source distributed analytics engine designed to provide a SQL interface and multi-dimensional analysis (OLAP) on large-scale datasets. It provides sub-second query latency on trillion-record datasets.

## APIs

### Apache Kylin REST API
The Kylin REST API provides endpoints for cube management, model management, job management, SQL queries, table management, and user authentication, along with JDBC and ODBC drivers for SQL access.
